Analytics Integration

How to add privacy-first page view tracking to the BMFFFL site using Vercel Analytics. No cookies, no consent banners.

Current Status

Not yet integrated. Pending Vercel deployment. Once the site is deployed to Vercel, follow the steps below — it takes under 5 minutes.

What Analytics Captures

Vercel Analytics is privacy-first — no cookies, no personal data, no GDPR consent required.

Page Views

Every page navigation is tracked, including client-side route changes via the Next.js router.

Unique Visitors

Distinct visitor counts per page and per session — without cookies or fingerprinting.

Top Pages

Ranked list of the most-visited pages, filterable by time window (24h, 7d, 30d).

Referrers

Where visitors come from — direct, search engines, social media, or external links.

Peak Usage Times

Hour-by-hour and day-of-week traffic patterns to understand when the league is most active.

Integration Steps

1

Install the package

bash
npm install @vercel/analytics
2

Import Analytics in _app.tsx

typescript
import { Analytics } from '@vercel/analytics/react';
3

Add <Analytics /> to the component tree

typescript
export default function App({ Component, pageProps }: AppProps) {
  return (
    <>
      <Component {...pageProps} />
      <Analytics />
    </>
  );
}

Place it inside the root component alongside your Layout wrapper.

4

Deploy to Vercel

Once deployed, the Analytics dashboard appears automatically in the Vercel project settings. No additional configuration required.
